This is version 2.3 of the nxslt.exe and NxsltTask for MSBuild and NAnt.

Changes:

o improved error reporting
o options files (@file option)
o XHTML output mode (-xhtml option)
o XSLT 2.0 character maps (-cm option)
o nxslt and NxsltTask are now debuggable
o resolving XInclude for XSLT stylesheets is now off by default
o -xslxi option to turn XInclude for XSLT stylesheets on
o bug fixes

See http://www.xmllab.net/nxslt for more info.
