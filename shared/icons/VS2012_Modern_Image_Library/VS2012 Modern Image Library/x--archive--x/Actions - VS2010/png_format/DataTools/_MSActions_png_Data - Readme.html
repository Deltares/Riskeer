Visual Studio Image Library: Actions (.png Format, DataTools)

32-Bit Art
This is 32-bit art that contains an alpha channel for transparency.

Use Restrictions
These images are commands relevant to the display of, and actions within, the XSDSchema designer.  As part of a visual language, they should be used in cases in which the meaning is consistent to the usage in that designer as described below.
