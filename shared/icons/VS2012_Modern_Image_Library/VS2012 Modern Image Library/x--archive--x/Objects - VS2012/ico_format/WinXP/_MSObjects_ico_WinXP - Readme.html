Visual Studio Image Library: Objects (.ico Format, Windows XP style)

Icon art
This filetype may contain more than one image size or color depth, or both. They are typically used in controls that offer variations in viewing style; a listbox which allows the user to choose between "small icon view" and "large icon view" is an example. The appropriate image size and color depth is displayed by the control. For all icon files, the best way to preview is through Windows Explorer; they do not show all variations within the file when displayed in the browser.
